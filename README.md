## macOS Binaries

- `dinerod` - Full node daemon
- `dinero-cli` - Command-line RPC client
- `dinero-miner` - CPU miner
- `dinero-qt.app` - GUI wallet application

### Verification

Verify downloads with SHA256 checksums:
```bash
cd mac
shasum -a 256 -c SHA256SUMS.txt
```

## Features

- **BIP86 Taproot-only** - Modern, privacy-preserving addresses
- **BIP39 mnemonic** - 12-word seed phrase backup
- **SHA256 Proof-of-Work** - Real mining, real security
- **PSBT support** - Hardware wallet compatible

## Quick Start

1. Download binaries for your platform
2. Run `dinerod` to start the node
3. Run `dinero-qt.app` for GUI wallet
4. Create wallet with 12-word seed phrase
